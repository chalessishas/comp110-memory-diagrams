## 匹配算法

```
用户向量 U = (d1..d10) ∈ [3, 9]^10
对每个角色 C_i:
  sim_i = cosine(U, C_i)          # 余弦相似度
主匹配 = argmax sim
次匹配 = 第二名
差距 = sim[主] - sim[次]

展示：
  - 主匹配 + 匹配度（百分比）
  - 次匹配 + "你在 X 和 Y 之间摇摆"（对冲 Barnum effect）
  - 若 max sim < 0.85 → "你是一个鸣潮尚未创造的原型"（长尾稀有结果）
  - 不匹配角色（argmin sim）+ "你最不像谁"（对自我认知价值高）
```

## 设计备注

- 30 题布局沿用 YSTI 范式，保持 10 维 × 3 题的干净映射
- 所有"纳西妲""钟离""璃月""尘歌壶""树脂""圣遗物""冒险者协会"等原神术语都已替换为鸣潮等效物（守岸人/忌炎/今州/共鸣居所/波片/声骸/公告板）
- 标注"虚构台词"的三处（Q10/Q13/Q26）：游戏内没有这些原文，是为了测试维度人工拟写的。上线前需替换为真实游戏台词，否则鸣潮玩家一眼拆穿
- 角色坐标为手工标注，基于角色传记、剧情站位和技能设定；未经玩家共识验证，只是 v0.1 草案
