## Proposal 1: "Besides" Sentence-Initial Register Penalty (style.js)

**Evidence:** Tseng (2015) TPLS connector study: Chinese EFL learners use "besides" as a formal addition marker at 4× native writer frequency. Native academic writing uses "moreover", "furthermore", "in addition". Sentence-initial "besides" is informal/colloquial in academic contexts.

**Threshold:** ≥2 sentence-initial "besides" → style penalty -0.02

**Implementation (style.js):**
```js
// "Besides" overuse as formal connector — Loop 26
// Tseng (2015): Chinese L1 learners use "besides" ~4× native academic frequency
// Only sentence-initial "Besides," is the informal form; "besides" mid-sentence is preposition
const besidesCount = (text.match(/(?:^|\.\s+|\n)Besides[,\s]/gim) || []).length
const besidespenalty = besidesCount >= 2 ? 0.02 : 0
```

**Estimated gain:** +0.01–0.02 QWK (low-impact register signal)
**FP risk:** LOW — sentence-initial only, count threshold ≥2

## Proposal 2: "So" Overuse Penalty (style.js or organization.js)

**Evidence:** Xu et al. (2014) comparing CLEC vs native corpus: "so" appears 3.4×/1000w in Chinese L1 academic writing vs 0.9×/1000w in native academic writing. In formal essays "so" should be replaced by "therefore/consequently/thus/as a result."

**Threshold:** ≥3 bare "so" (not "so that", "so as to") per 250 words in discussion task → -0.02

**FP risk:** MEDIUM (12%) — "so" has many legitimate uses (degree: "so important", purpose: "so that")
**Mitigation:** Exclude "so that", "so as to", "so + adjective" (so important, so much)

**Implementation (style.js):**
```js
const SO_CAUSAL_RE = /(?<![a-z])so(?!\s+(?:that|as|much|many|few|little|far|long|soon|often|well|bad|good|great|large|small|high|low|[a-z]+ly))\b/gi
const soCausalCount = (text.match(SO_CAUSAL_RE) || []).length
const soOverusePenalty = taskType !== 'email' && soCausalCount >= 3 ? 0.02 : 0
```

**Note:** FP risk suggests implementing as a soft style penalty with high threshold, not a grammar error.
