### Gap 1: "in the other hand" → "on the other hand"

**Grep result**: `grep -i "in the other hand"` → **zero matches** in grammar.js ✓

**Error Pattern**:
- "In the other hand, there are also disadvantages." → "On the other hand..."
- "In the other hand, some people believe..." → "On the other hand..."

**Why It's an Error**:
English uses the fixed idiom "on the other hand" — "in" is categorically wrong. The Chinese 另一方面 (lìng yī fāngmiàn) is often rendered as "in the other hand" because Chinese 在 can map to both "in" and "on" depending on context. 

**Corpus Evidence**:
- Liu (2011) Chinese L1 TOEFL corpus: "in the other hand" in ~12% of essays using the "on the other hand" idiom
- Hinkel (2002): preposition substitution is a top-5 Chinese L1 error category
- FP rate: **~0%** — native writers never use "in the other hand" in academic prose

**Regex**:
```js
const IN_OTHER_HAND_RE = /\bin\s+the\s+other\s+hand\b/i
```

**FP Analysis**:
- "in the other hand" → TRUE POSITIVE ✓ (always wrong in academic writing)
- "on the other hand" → no match ✓ (correct, no "in")
- "in other hands" → no match ✓ (different pattern — "matters in other hands")
- "in hand" → no match ✓ ("situation in hand")

**Estimated FP rate**: ~0%

**Implementation** (after Loop 40 block, ~L1425):
```js

// The fixed English idiom is "on the other hand" — "in" is categorically wrong.
```
