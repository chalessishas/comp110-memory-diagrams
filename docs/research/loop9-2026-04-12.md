## Gap Analysis: What's Still Missing

### Gap 1: Discourse Connective Quality vs. Quantity
**Current state:** `organization.js` counts distinct markers and categories, with an inferential/additive ratio bonus.

**Gap:** The system doesn't distinguish *where* in the essay connectors appear. ETS research (Ferris 1994, Crossley 2020) shows Score-5 writers front-load conclusion markers at paragraph starts ("Therefore, ...") and transition between body paragraphs with adversative connectors ("However, this perspective overlooks..."), while Score-4 writers scatter connectors mid-sentence.

**Proposed feature: Paragraph-initial marker bonus**
- Detect connectives at sentence start (first 3 words of a sentence)
- Reward paragraph-opening discourse markers: +0.04 for ≥2 paragraph-opening transitions
- Est. ROI: 0.04–0.06 QWK
- False positive risk: Low (the pattern is straightforwardly detected)

```js
// Count sentence-initial discourse connectors
const sentences = text.split(/[.!?]+/).map(s => s.trim()).filter(Boolean)
const PARA_INIT_MARKERS = /^(However|Furthermore|Moreover|Therefore|Consequently|Nevertheless|In contrast|By contrast|Admittedly|In addition|Additionally)/i
const paraInitCount = sentences.filter(s => PARA_INIT_MARKERS.test(s)).length
const paraInitBonus = paraInitCount >= 3 ? 0.06 : paraInitCount >= 2 ? 0.04 : 0
```

### Gap 2: Sentence-Level Syntactic Maturity (Mean Length of T-unit proxy)
**Current state:** `style.js` measures sentence length variance but not average length or T-unit density.

**Gap:** SCA (Syntactic Complexity Analyzer) research (Lu 2010, 2012; used in ETS AES validation) shows Mean Length of T-unit (MLT) is one of the top 3 syntactic complexity features. Score-5 TOEFL essays average 14–18 words/sentence; Score-3 averages 10–13 words/sentence. The current system has no explicit sentence length floor/ceiling reward.

**Proposed feature: Mean sentence length bonus**
```js
const meanSentLen = tokens.length / sentences.length
// Too short (<10 avg): likely choppy, underdeveloped
// Sweet spot (13-20): well-developed, complex
// Too long (>25 avg): may indicate run-ons or padding
let sentLenBonus = 0
if (meanSentLen >= 14 && meanSentLen <= 22) sentLenBonus = 0.04
else if (meanSentLen >= 12) sentLenBonus = 0.02
```
- Est. ROI: 0.03–0.05 QWK
- False positive risk: Low — long sentences are separately checked by grammar.js run-on detector

### Gap 3: Idiomatic Phrasal Verb Usage
**Current state:** No detection of phrasal verbs or idiomatic expressions.

**Gap:** ETS rubric explicitly distinguishes Score-4 ("appropriate word choice") from Score-5 ("precise, idiomatic word choice"). Research on idiomatic language in L2 writing (Nesselhauf 2003; Wray 2002) shows that high-scoring L2 writers use more phrasal verbs naturally: "point out", "account for", "bring about", "result in", "take into account". Chinese L1 writers tend to avoid them and default to formal Latin synonyms exclusively.

**Challenge:** Detecting phrasal verb presence is straightforward; detecting *correct* collocation is hard in pure JS.

**Safe proxy:** Count distinct phrasal verbs from a positive list. Presence of ≥3 different phrasal verbs → small bonus.

```js
const PHRASAL_VERBS = [
  /\bpoint(s|ed|ing)? out\b/i,
  /\baccount(s|ed|ing)? for\b/i,
  /\bbring(s|ing)?\b.{0,5}\babout\b|\bbrought about\b/i,
  /\bresult(s|ed|ing)? in\b/i,
  /\btake(s|n|taking)?\b.{0,5}\binto account\b|\btook into account\b/i,
  /\bcarry(ing|ied)? out\b|\bcarried out\b/i,
  /\bset(ting)? (up|out|aside)\b/i,
  /\bcall(s|ed|ing)? for\b/i,
  /\blead(s|ing)?\b.{0,5}\bto\b|\bled to\b/i,
  /\bbuild(s|ing)?\b.{0,5}\bon\b|\bbuilt on\b/i,
]
const phrasalHits = PHRASAL_VERBS.filter(re => re.test(text)).length
const phrasalBonus = phrasalHits >= 3 ? 0.05 : phrasalHits >= 2 ? 0.03 : 0
```
- Est. ROI: 0.03–0.05 QWK
- False positive risk: Low — phrasal verbs in this list are unambiguous
- Where to add: `vocabulary.js` (lexical sophistication) or `style.js` (register/idiom)

### Gap 4: Contrastive Stance Elaboration
**Current state:** We detect hedging (tier-1 and tier-2) but not the follow-through after a concessive.

**Gap:** Score-5 writers typically structure arguments as: [claim] + [concession] + [rebuttal]. A concessive marker ("Although/Admittedly/While it is true that X,") followed by substantive elaboration is a strong Score-5 signal. Score-4 writers often use the concession structure but don't develop the rebuttal.

**Difficulty:** Detecting the quality of elaboration after a concessive is semantically complex. A safe proxy: detect concessive-then-rebuttal within 2 sentences ("Admittedly...[sentence]...However/Nevertheless/That said...").

**Assessment:** Moderate complexity to implement correctly. Low priority vs. Gap 1-3.
