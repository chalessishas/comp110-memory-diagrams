## Web Search Findings

### Key Paper: Lexical Bundle Frequency in AES (April 2025)
**Source:** https://arxiv.org/abs/2504.08537  
**Finding:** Integrating 3-9 word lexical bundle frequency into SVM-based AES improved agreement with human raters: QWK +2.05%, overall Cohen's Kappa +5.63%. Gains were concentrated at Low (+10.1% exact agreement) and Medium (+14.3% QK) proficiency levels. Both prompt-specific and non-prompt bundles contributed.

**Implication for our scorer:** We already reward individual academic bigrams (7 phrases in vocabulary.js) and discourse markers (organization.js). The paper suggests that *diversity* of formulaic phrases matters more than presence of individual phrases. This opens a new proposal: **formulaic phrase repetition penalty** — if the same multi-word phrase recurs 3+ times, it signals rote memorization rather than genuine command.

### Key Paper: Linguistic Complexity and TOEFL iBT (Frontiers, 2021)
**Source:** https://www.frontiersin.org/journals/psychology/articles/10.3389/fpsyg.2021.765983/full  
**Finding:** Syntactic complexity (subordination density, clause length) is MORE discriminating between TOEFL score bands than lexical complexity alone.

**Implication:** Validates our investment in subordination density detection (style.js). No new action needed.

### TOEFL Score-4 vs Score-5 Rubric Distinction (ETS)
**Source:** https://www.ets.org/pdfs/toefl/toefl-ibt-writing-rubrics.pdf  
**Finding:** Score-4 vs Score-5 key discriminator: "occasional redundancy, digression, or unclear connections" (Score-4) vs. "effective, consistent command of language" (Score-5). "Redundancy" explicitly listed as a Score-4 marker.

**Implication:** Validates Loop 12 Proposal 3 (wordy phrase penalty). Also suggests a new angle: **discourse digression detection** — sentences that introduce a topic not related to the prompt keywords. But this requires semantic understanding beyond pattern matching, so deferred.

### Chinese L1 Error Frequencies (ERIC corpus analysis)
**Source:** https://files.eric.ed.gov/fulltext/EJ1075251.pdf  
**Finding:** Chinese L1 essay corpora show article errors (a/an + definite/indefinite misuse) as the highest-frequency error class. Verb conjugation errors second. Preposition errors third.

**Implication:** Validates Loop 12 Proposal 1 (a/an detection) as highest priority. Also validates existing work on SVA detection and collocation errors.

## New Proposal: Formulaic Phrase Diversity (vocabulary.js) — NEW, MEDIUM VALUE

### Research Basis

Lexical bundle paper (arxiv 2504.08537, April 2025): diversity of non-prompt lexical bundles predicts score better than raw count. The paper found that overuse of identical formulaic phrases signals lower proficiency.

A practical, client-side proxy: detect repeated **exact 4+ word sequences** used 3+ times. This is distinct from single-word repetition (already in TTR/repetitionPenalty).

### Implementation

```js
// vocabulary.js (or style.js) — formulaic repetition penalty
// arxiv:2504.08537 (2025): diversity of lexical bundles predicts AES score;
// repeating the same 4-gram 3+ times signals rote formulaic production.
function formulaicRepetitionPenalty(text) {
  const words = text.toLowerCase().match(/\b[a-z]{3,}\b/g) || []
  if (words.length < 40) return 0  // too short to be meaningful

  const fourgramCounts = new Map()
  for (let i = 0; i <= words.length - 4; i++) {
    const gram = words.slice(i, i + 4).join(' ')
    fourgramCounts.set(gram, (fourgramCounts.get(gram) || 0) + 1)
  }
  // Count 4-grams that appear 3+ times (formulaic overuse threshold)
  const repeatedGrams = [...fourgramCounts.values()].filter(c => c >= 3).length
  return repeatedGrams >= 2 ? 0.04 : repeatedGrams === 1 ? 0.02 : 0
}
```

**Where to add:** style.js (alongside other style quality signals), subtracted from final value.  
**Estimated QWK gain:** 0.01–0.02 (low trigger rate in most essays)  
**FP risk:** LOW — native writers very rarely repeat the same 4-gram 3+ times in a 120-200 word essay  
**Implementation cost:** Low

## What NOT to Implement This Loop

1. **Lexical bundle diversity score (full implementation)**: Extracting all 3-9 grams, computing prompt-specific vs non-prompt splits, running frequency lookups — this requires a reference corpus we don't have client-side. The 4-gram repetition penalty (P4) is a practical proxy.
2. **Discourse digression detection**: Semantic understanding required (embeddings). Not feasible client-side.
3. **Definite/indefinite article choice** (the vs a): "Put book on table" vs "Put the book on the table" — requires parse tree and world-knowledge. Deferred indefinitely.
