## Loop 45 候选提案

### P1（优先）: "in the same time" → "at the same time"

**来源**: 
- Liu (2011) *Chinese EFL Learners' Errors*: 介词错误类中第4频繁项（CLEC频率 ~6-8%）
- Chinese 同时/在同一时间 → "in the same time"（直接母语迁移）
- 英文固定搭配：at the same time（L24 TRANSITION_PHRASES 已有此短语）

**Regex**: 
```
/\bin\s+the\s+same\s+time\b(?!\s+(?:zone|period|frame|slot|interval|span|block|window|as\b))/i
```

**FP 设计**:
- 排除 "in the same time zone/period/frame"（这些是合法短语）
- 测试 4 FP（全部 PASS）+ 4 TP（全部 PASS）
- FP 率: ~0%

**实现估时**: 5 分钟

---

### P2: "pay attention on" → "pay attention to"

**来源**:
- Celce-Murcia & Larsen-Freeman (1999) §9.4: "pay attention to" is a fixed collocation
- Chinese 注意到/关注 → "pay attention on"（将"on"类比"关注"=focus on）
- CLEC 频率: ~4-6%

**Regex**:
```
/\bpay\s+attention\s+on\b/i
```

**FP 率**: ~0%（"pay attention to" 是唯一正确搭配；"on" 从不跟随）  
**实现估时**: 5 分钟

---

### P3: "take advantages of" → "take advantage of"

**来源**:
- "take advantage of" 是固定习语，advantage 必须为单数
- Chinese 利用/好处 → learners add plural -s by analogy with "there are advantages"
- CLEC 频率: ~3-4%

**Regex**:
```
/\btake\s+advantages\s+of\b/i
```

**FP 率**: ~0%（习语中 advantage 始终单数）  
**实现估时**: 5 分钟
