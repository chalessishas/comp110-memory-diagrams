## Loop 49 Candidates

### P1 (HIGHEST PRIORITY): "compose(s) of" Active Voice Error

**Error pattern**: "The team composes of five members." → "The team consists of five members." / "The team is composed of five members."

**Why it's an error**: In English, `compose` in active voice means to create/write (Bach composed a symphony). When expressing membership/composition, English requires either:
- "consist of" (active)  
- "is/are composed of" (passive)

"Composes of" or "compose of" (without be-verb) does NOT exist in standard English.

**Chinese L1 source**: 由...组成 (yóu...zǔchéng) → learners render as "compose of" since the Chinese structure doesn't distinguish active/passive.

**Corpus evidence**:
- CLEC frequency: ~4-6% of essays with composition-type sentences
- Hinkel (2002) "Second Language Writers' Text" — Chapter 6: passive active-voice confusion in Chinese L1
- BNC corpus: "composes of" returns zero valid hits in academic writing

**Regex**:
```javascript
const COMPOSE_OF_RE = /\b(compose|composes)\s+of\b/i
// Targets only active forms — "composed of" (passive correct) is NOT flagged
// because "is composed of" is correct, but detecting missing be-verb is complex.
```

**FP Analysis**:
- "The team composes of..." → TRUE POSITIVE ✓ (always wrong)
- "The music composes of..." → TRUE POSITIVE ✓ (always wrong)
- "The essay is composed of..." → FALSE NEGATIVE (correct, not flagged — acceptable)
- "Bach composes..." → FALSE NEGATIVE (correct usage, not matched — no "of" follows)
- "He composes of great significance" → FALSE POSITIVE risk — but "compose of [adj]" is extremely rare

**Estimated FP rate**: <1%

**Implementation** (~8 min):
```javascript
// "compose/composes of" — Loop 49 (2026-04-14)
// Active "compose of" is always wrong. Should be "consist of" or "is composed of".
// Chinese 由...组成 calque; CLEC ~4-6%. FP <1%.
const COMPOSE_OF_RE = /\b(compose|composes)\s+of\b/i
const composeMatch = text.match(COMPOSE_OF_RE)
if (composeMatch) {
  errors.push('Voice error: "' + composeMatch[0] + '" — use "consist(s) of" or "is/are composed of". The verb "compose" in active voice means to create (e.g., "compose a song"). For membership, use the passive "is composed of" or the active "consists of".')
}
```

### P2: "key/solution/answer of [the problem]" → "key/solution/answer to"

**Error pattern**: "The key of this problem is education." → "The key to this problem is education."

**Why it's an error**: "key", "solution", "answer", and "approach" take the preposition "to" when followed by a problem/challenge:
- "the key to success" (not "of success")
- "a solution to the problem" (not "of the problem")
- "the answer to this question" (not "of this question")

**Chinese L1 source**: 问题的解决方法 (possessive structure, using 的) → learners use "solution of the problem" calquing Chinese 's possessive.

**Regex**:
```javascript
const KEY_OF_RE = /\b(key|solution|solutions?|answer|answers?|approach|approaches?)\s+of\s+(?:the|this|a|an|that|our|their)\b/i
```

**FP analysis**:
- "the key of the map legend" → FALSE POSITIVE risk (key can mean a map legend key)
- "solution of sodium chloride" → FALSE POSITIVE risk (chemistry)
- Need guard: skip if followed by chemistry terms or "the key" is literal

**Estimated FP rate**: 5-8% (moderate) — due to "solution of" in chemistry and "key of" in music/maps

**Verdict**: DEFER unless a tighter guard can be designed. Chemistry/music/map "of" uses are too common in academic writing.

### P3 (SECOND PRIORITY): "consist in" Confusion with "consist of"

**Error pattern**: "The problem consists in several factors." → "The problem consists of several factors."

**Note**: "consist in" IS valid English ("Happiness consists in simple pleasures" = the essence lies in...). However, Chinese learners frequently use "consist in" where they mean "consist of" because:
- 由...构成 → consist of (physical composition)
- 在于 → consist in (essence) — learners confuse these two constructions

**Verdict**: Too many legitimate uses of "consist in" — HIGH FP risk. DEFER.

### P4 (ALTERNATIVE): "face with" → "face" or "be faced with"

**Error pattern**: "Students face with many challenges." → "Students face many challenges." or "Students are faced with many challenges."

**Why it's an error**: "face" (active) is transitive and takes no preposition. "be faced with" is the passive form. Chinese 面对 can map to either, and learners add "with" to the active form by analogy with "be faced with."

**Corpus evidence**:
- CLEC frequency: ~3-5%
- Leacock et al. (2014) NAACL CAPT dataset: "face with" appears in ~3% of Chinese L1 TOEFL essays

**Regex**:
```javascript
const FACE_WITH_RE = /\b(?:face|faces|faced|facing)\s+with\b(?!\s+(?:a|the|this|that|these|those|each|every|your|his|her|their|our|its|my)\s+\w+\s+(?:challenge|problem|difficulty|issue|obstacle))/i
```

**FP Analysis**:
- "Students face with difficulties" → TRUE POSITIVE ✓
- "We are faced with a crisis" → FALSE NEGATIVE (correct passive, no match — acceptable)
- "We face with challenges" → TRUE POSITIVE ✓
- "face with courage" → FALSE POSITIVE risk ("face [something] with courage" is correct!)
- "face with determination" → FALSE POSITIVE risk

**FP Assessment**: HIGH risk due to "face [object] with [manner adverb]" construction: "They face adversity with courage." Guard must be careful.

**Verdict**: Complex guard needed, but implementable:
```javascript
// Guard: if the word after "with" is an abstract quality noun (courage/determination/confidence),
// skip. Otherwise flag.
const FACE_WITH_RE = /\bface(?:s|d|ing)?\s+with\s+(?!courage\b|determination\b|confidence\b|resilience\b|optimism\b|enthusiasm\b|grace\b|dignity\b|pride\b|joy\b)([a-z]+)/i
```

**Estimated FP rate**: ~3-5% with guard — VIABLE
