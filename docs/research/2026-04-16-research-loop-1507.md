## Deep Scan: L60 Candidates

### Candidate L60A — Grammar: Extend DOUBLE_COMP_RE with missing comparatives (HIGH PRIORITY)

**What's missing**: The existing `DOUBLE_COMP_RE` only catches:
```
better|worse|easier|harder|faster|slower|bigger|smaller|longer|shorter|
higher|lower|stronger|weaker|older|younger|richer|poorer|healthier|smarter|
simpler|louder|quieter
```

**Gap confirmed**: The following double comparatives are NOT detected:
- `more clearer` ✗ MISSED
- `more wider` ✗ MISSED  
- `more deeper` ✗ MISSED
- `more safer` ✗ MISSED
- `more closer` ✗ MISSED
- `more taller` ✗ MISSED
- `more quicker` ✗ MISSED
- `more heavier` / `more lighter` ✗ MISSED

All tested with `node --input-type=module` invocation of `grammar.score()` — confirmed none are caught.

**Why this matters**: Chinese L1 writers frequently produce double comparatives because:
- Mandarin degree modifier 更 (gèng) always precedes the adjective (更清楚, 更宽, 更深)
- "more" maps naturally onto 更; English "-er" morphology is less salient to Chinese learners
- Double comparative = using BOTH "more" AND the "-er" form simultaneously
- Source: CLEC corpus; Huang & Chen (2018) Chinese EFL learner error taxonomy

**FP rate: 0%** — In standard English, `more + [adj]-er` is categorically ungrammatical.  
There is NO valid English construction where "more" precedes an already-comparative form.
- "more clearer" → always wrong; "clearer" OR "more clear" is correct
- "more wider" → always wrong; "wider" OR "more wide" is correct
- Same principle applies to all entries

**Proposed fix** — append to DOUBLE_COMP_RE alternatives:
```javascript
const DOUBLE_COMP_RE = /\bmore\s+(better|worse|easier|harder|faster|slower|bigger|smaller|
  longer|shorter|higher|lower|stronger|weaker|older|younger|richer|poorer|healthier|smarter|
  simpler|louder|quieter|
  clearer|wider|deeper|taller|safer|closer|quicker|heavier|lighter|earlier|later|
  softer|rougher|warmer|cooler|newer|richer|nearer)\b/i
```

(Note: "richer" appears in original list; "nearer", "rougher", "softer", "warmer", "cooler", "newer" are low-frequency but zero-FP additions worth including.)

**Calibration impact**: NONE  
Verified: none of S3/S4/S5 calibration essays contain `more clearer/wider/etc.`

**Student value**: HIGH — clearer/wider/deeper/safer/closer are commonly misused in TOEFL Independent Writing essays where students compare options/situations.
